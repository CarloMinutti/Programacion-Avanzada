II · ¿Qué es un decorador?

Un decorador es una función que envuelve a otra
para añadirle algo sin tocarla.

El símbolo @ que verás antes de property es azúcar sintáctico — un atajo elegante para algo más largo. Vamos por partes:

¿Qué es "azúcar sintáctico"?
Una forma de escribir algo más corta y bonita que hace exactamente lo mismo que la forma larga. No añade poder nuevo — solo lo hace más legible. Igual que x += 1 es azúcar para x = x + 1, el @gritar es azúcar para saludar = gritar(saludar).
